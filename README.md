# Desarrolla una implementación completa del algoritmo QuickSort en Java que incluya:

- Implementación recursiva.
- Uso de partición con pivote como último elemento.
- Medición de tiempo con System.nanoTime().
- Conteo de comparaciones.
- Evaluación de tres escenarios: aleatorio, ordenado e inverso.
- Pruebas con tamaños 1000, 2000, 5000 y 10000.
- Impresión en formato CSV para generar gráfica.

El objetivo es analizar empíricamente la complejidad temporal del algoritmo y comparar los escenarios.
